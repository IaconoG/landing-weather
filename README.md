# Landing Weather - React + Vite Template

Personal project template for a weather app built with React, TypeScript, and Vite. It includes features such as location search, current weather display, and a responsive design.

## Current Status

In process of development. The project is being build.

## Tech Stack

- React
- TypeScript
- Vite
- Zustand (state management)

## Run Locally

1. Clone the repository:
   ```bash
   git clone
   ```
2. Navigate to the project directory:
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
5. ```bash
   npm run dev
   ```

## Notes

This is a personal project and is currently in development. The codebase may undergo significant changes, and some features may not be fully implemented yet.
